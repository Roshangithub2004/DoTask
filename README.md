# Installation & Setup

1. Clone the repository

    -> git clone git remote add origin https://github.com/Roshangithub2004/DoTask.git


2. Install backend dependencies

    cd backend
    npm install


3. Start the backend server
    npm run dev


4. Install frontend dependencies
    cd frontend
    npm install
    npm run dev
